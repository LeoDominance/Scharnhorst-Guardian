# Scharnhorst-Guardian

A robust system monitoring and security tool designed for real-time anomaly detection and resource optimization.

## Features
- Real-time monitoring of system processes.
- Anomaly detection to identify suspicious resource consumption.
- Whitelist support to exclude trusted applications.
- Automated security alerts.

## How to use
1. Ensure you have Python installed.
2. Install requirements: `pip install psutil`
3. Run the tool: `python guardian.py`
